Optimization


 This project contains examples for optimizing circuits.
 This can be performed for all analogue simulations.

power_matching.sch  -  most simple example for a circuit optimization
noise_figure.sch  -  optimization with two variables
diplexer.sch  -  a more complex example: creating a diplexer out of a standard filter
triplexer.sch  -  an even more complex example: creating a triplexer out of standard filters
model.sch  -  the typical optimization: fitting simulation to measurement
stub_filter.sch  -  example with goals for different frequency ranges
inverter.sch  -  an example with parameter sweep
broadband_matching.sch  -  a quite complex example with many parameters
